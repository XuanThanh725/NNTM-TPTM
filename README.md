# 📖 1. Giới thiệu hệ thống

Hệ thống nhận diện và phân loại vật nuôi là ứng dụng sử dụng trí tuệ nhân tạo (AI) kết hợp mô hình YOLOv8 nhằm phát hiện, nhận diện và thống kê các loại vật nuôi từ hình ảnh, video hoặc camera thời gian thực.

Hệ thống hỗ trợ:

- Nhận diện vật nuôi theo thời gian thực.
- Phân loại nhiều loại vật nuôi khác nhau.
- Đếm số lượng vật nuôi.
- Tự động chụp ảnh khi phát hiện.
- Lưu lịch sử nhận diện vào CSV.
- Xuất báo cáo Excel.
- Hỗ trợ Webcam, Camera IP và ESP32-CAM.
- Dashboard thống kê trực quan.
